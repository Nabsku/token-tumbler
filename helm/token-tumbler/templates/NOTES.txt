1. Get the application logs by running these commands:
  kubectl logs -f deployment/{{ include "token-tumbler.fullname" . }}

2. Verify the ConfigMap contains your repositories:
  kubectl get configmap {{ include "token-tumbler.fullname" . }} -o yaml

3. The application reads these environment variables from the Secret:
   - GITLAB_TOKEN
   - GITLAB_URL
   - TOKEN_TUMBLER_INTERVAL (optional, default: 5m)
   - APPROLE_ID (if using Vault)
   - APPROLE_SECRET (if using Vault)

   These are stored in the Secret: {{ if .Values.existingSecret }}{{ .Values.existingSecret }}{{ else }}{{ include "token-tumbler.fullname" . }}{{ end }}
