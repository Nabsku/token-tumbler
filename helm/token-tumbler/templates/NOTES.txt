  kubectl get configmap {{ include "token-tumbler.fullname" . }} -o yaml

3. The application requires the following environment variables to be set:
   - GITLAB_TOKEN
   - GITLAB_URL
   - TOKEN_TUMBLER_INTERVAL (default: 5m)
   - APPROLE_ID (if using Vault)
   - APPROLE_SECRET (if using Vault)
